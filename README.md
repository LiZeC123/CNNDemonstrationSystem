### Flask

#### Request 对象

通过Request的以下成员变量获得请求参数

参数    | 含义
--------|-------------------------
args    | 查询字符串
data    | 字符串形式的请求数据
files   | 上传的文件
form    | 表单数据
values  | args和form的数据
json    | 解析后的JSON数据
